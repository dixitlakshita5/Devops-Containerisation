# Devops-Containerisation

This repository consists of all the Lab experiments of Containerisation and DevOps.

## Lab Experiments

- [Experiment 0](./LAB/Experiment-0/README.md) 
    - This experiment demonstrates the installation of Docker Desktop on macOS and Git for integrating and managing version-controlled work efficiently as a Mac user.
- [Experiment 1](./LAB/Experiment-1/README.md)
    - To study and compare Virtual Machines and Containers by deploying an Ubuntu-based Nginx server in both environments and analyze differences in performance, resource utilization, and operational efficiency on a macOS (Apple Silicon M2) system.
- [Experiment 2](./LAB/Experiment-2/README.md)
    - To install and configure Docker, pull Docker images, run containers, and manage the container lifecycle using Docker commands.
- [Experiment 3](./LAB/Experiment-3/README.md)
    - To deploy and analyze NGINX using different Docker base images (official, Ubuntu, and Alpine) in order to understand image layers, size differences, performance, security, and real-world containerized use cases.
